## Featured Project

### [Selenium Test Automation Framework](https://github.com/mtiengo/selenium-test-automation-framework)

A modern, scalable test automation framework built from scratch with:
- **Page Object Model (POM)** architecture with instance-based design.
- **Thread-safe parallel execution** via `ThreadLocal`-backed `DriverManager`.
- **Cross-browser support** (Chrome, Firefox, Edge) via TestNG `@Parameters`.
- **ExtentReports** integration with custom `ITestListener`.
- **TestNG suites** with smoke groups and Maven Surefire integration.
- **Java 21 + Selenium 4 + TestNG + WebDriverManager**.
- 12+ test scenarios covering forms, tables, React components, and file uploads.
- Built-in screenshot capture on test failures.
